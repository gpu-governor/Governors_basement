Getting Started with ncurses

1. Creating Your First ncurses Program

Create a main.c or main.cpp file and open it in your IDE of choice (I personally use Vim and Micro). Copy the code below to your file:


#include <ncurses.h>

int main() {
    initscr();
    printw("C is the fastest language");
    refresh(); // Update the screen with the printed output
    getch();   // Wait for user input
    endwin();  // End ncurses mode
    return 0;
}

If you compile and run the program, you would see nothing. To compile, if you are using Visual Studio or a similar IDE, add the ncurses library to the linker and click the run button. However, since I am coding in my Linux terminal, I would just run:


gcc main.c -o main -lncurses
./main

The program ran successfully but showed no output. The reason is that we did not call the refresh() function after our printw().
The refresh() function is responsible for updating the screen with whatever is to be displayed. Even after adding refresh(), you still wouldn't see anything because the program closes immediately, and the human eye is too slow to see what was displayed.

To ensure the program waits for us to finish seeing the output, we should add the getch() function after the refresh(). This function getch() (get char) stops and waits for the user to press any key and then stores the key value in memory before proceeding with the program, which then ends.

So now if you add refresh() and getch() to our main.c, you should see the output.

Here is another simple "Hello World" ncurses program:


#include <ncurses.h>

int main() {
    initscr();            // Start ncurses mode
    printw("Hello, world!"); // Print message to the screen
    refresh();            // Refresh the screen to show the output
    getch();              // Wait for user input
    endwin();             // End ncurses mode
    return 0;
}
  • initscr() is called first to initialize ncurses.
  • printw() is used to print a string to the screen.
  • refresh() updates the screen with the printed output.
  • getch() waits for the user to press a key before exiting.
  • endwin() ends ncurses mode.

Conclusion

The ncurses library is powerful for creating text-based user interfaces. With basic functions like initscr(), printw(), getch(), and refresh(), you can start building interactive programs in a terminal. Experiment with these functions to get comfortable; we will explore more advanced features like windows, colors, and key handling as you progress.
